# 🚀 Desafio Root - Full Stack Project

Repositório central do projeto **Desafio Root**, estruturado com submódulos Git para separar as camadas de Backend e Frontend.

### 🏗️ Arquitetura do Projeto
* **[Backend](./backend/desafio-root-api):** API em .NET 8 com PostgreSQL.
* **[Frontend](./frontend):** SPA em React + Vite.

### 📥 Como Clonar o Projeto
Para baixar o repositório principal e todos os submódulos de uma só vez:

# Comando de clonagem recursiva
git clone --recursive [https://github.com/MROR21/desafio-root.git](https://github.com/MROR21/desafio-root.git)


### Caso você já tenha clonado de forma comum (sem o --recursive), as pastas estarão vazias. Para corrigir e baixar os códigos, execute:

git submodule update --init --recursive
